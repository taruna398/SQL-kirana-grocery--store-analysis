# Kirana Store Sales Analysis — SQL Project

## About the Project
Analyzed sales data from a local Kirana store
using PostgreSQL to find business insights.

## Tools Used
- PostgreSQL
- pgAdmin 4

## Dataset
- 200+ transactions
- January 2024 to December 2024
- Columns: Product, Category, Sales, Cost, Profit

## Business Questions Answered
1. Which month had the highest sales?
2. Which products sell the most?
3. Which category is most profitable?
...and 7 more insights
